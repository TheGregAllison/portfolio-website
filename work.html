React Meet App

The Meet App demonstrates the use of Google OAuth2 and uses the Google Calendar API to populate sample events. The Recharts data visualization library is used to show data regarding the frequency of event topics and locations. This project also integrated test automation frameworks Jest, Cucumber, and Puppetteer to exercise test and behavior-driven development methodologies.
